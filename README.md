# Modern: Luxe Blue Personal Portfolio

## Introduction

The **Modern: Luxe Blue Personal Portfolio** is a visually elegant and
highly functional personal portfolio built using React, Tailwind CSS,
and Framer Motion. This template is tailored for developers, designers,
and creatives who want a clean, responsive, and modern interface to
showcase their work and professional identity.

Whether you're applying for jobs, freelancing, or building your online
presence, this portfolio helps you stand out in a competitive market.

------------------------------------------------------------------------

## Why a Portfolio Matters

In today's digital-first world, a resume alone is not enough. A
portfolio acts as a career differentiator.

### Why portfolios matter:

-   They show, not just tell --- recruiters see live examples of your
    work.
-   They give you control over your personal branding.
-   They increase your chances of being remembered and contacted.
-   They are accessible 24/7, globally.

-   Resume header\

-   Cover letters\

## How to Edit Content

The portfolio uses modular React components:

-   `Hero.tsx` → Name, title, intro, social links\
-   `About.tsx` → Professional journey and timeline\
-   `Projects.tsx` → Project data via `projectsData.ts`\
-   `Skills.tsx` → Skills via `skillsData.ts`\
-   `Contact.tsx` → Contact details and form logic\
-   `Footer.tsx` → Social links and location

Update the `.tsx` files or the data files to customize your content.

## Running Locally

1.  Download the project as a ZIP file\

2.  Extract it to your desired location\

3.  Open the folder in VS Code\

4.  Open a terminal in the project directory\

5.  Run:

    npm install

6.  Then run:

    npm run dev

7.  Open your browser and navigate to:

    http://localhost:5173

------------------------------------------------------------------------

## Deployment

### Deploy with Netlify

1.  Go to netlify.com\

2.  Click "Add new site" → "Import from Git"\

3.  Connect your GitHub repository\

4.  Use the following build settings:

    Build command: npm run build\
    Publish directory: dist

5.  Click Deploy

You will receive a live production URL.

### Other Deployment Options

-   Vercel\
-   GitHub Pages\
-   Firebase Hosting\
-   Render
